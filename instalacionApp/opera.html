Instalar opera
# Descargar opera desde su página oficial

# Instalarlo sudo apt install ./Descargas/nombreDelPaqueteDeOpera.deb
Cambiar idioma
# En una terminal, buscar el lanzador de opera cd /usr/share/applications # Editarlo sudo nano opera.desktop # En la parte de Exec=opera %U # Reemplazar por Exec=opera --lang="es-mx" %U
Instalar el componente libffmpeg.so para reproducir videos
# En opera, abrir la pestaña "Acerca de" buscando el URL opera://about o con el boton en la esquina superior izquierda seleccionar "Ayuda -> Acerca de Opera"

# En la sección "Identificacion del Navegador" fijarse en los números que siguen a "Chrome/" ej: "Chrome/116"

# Buscar aquí la carpeta con los números que encontramos y descargar el archivo libffmpeg.so que está dentro de ella

# Nuevamente en "Acerca de" en la sección "Rutas" copiar la direccion de Instalar (ej: /usr/lib/x86_64-linux-gnu/opera)

# En una terminal, abrir la direccion obtenida cd /direccionObtenida # Crear una carpeta sudo mkdir lib_extra # y mover el archivo libffmpeg.so a la carpeta creada sudo mv /direccionDelArchivo/libffmpeg.so lib_extra
