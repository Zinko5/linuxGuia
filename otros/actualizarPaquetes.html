Actualizar paquetes
# Usar los comandos para actualizar paquetes sudo apt-get update
sudo apt-get upgrade
sudo apt update
sudo apt upgrade
sudo apt update # Es posible que salga un mensaje diciendo que hay paquetes no actualizados, para verlos usar apt list --upgradable # Nos saldra una lista de los paquetes no actualizados, para actualizarlos usar el siguiente comando con cada uno de los nombres de los paquetes que queremos actualizar sudo apt-get install --only-upgrade NOMBRE_DEL_PAQUETE # Si sale otro error, probablemente es un error relacionado con las keys o los repositorios
