Instalar Foo2zjs
# Foo2zjs es un driver para impresoras HP LaserJet

# Clonar el repositorio de Foo2zjs git clone https://github.com/koenkooi/foo2zjs # Instalar el driver make
make install
make install-hotplug # Si sale el siguiente error Error: system-config-printer-udev is installed!
 # Obedecer las instrucciones y quitarlo sudo apt-get remove system-config-printer-udev
