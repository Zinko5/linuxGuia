Comandos para servicios
Ver servicios y su estado (estado no confiable) service --status-all Ver servicios sin su estado ls /etc/init.d/ Verificar si un servicio está habilitado para iniciar automáticamente ls /etc/rc*.d/*nombreServicio *Si no se encuentran archivos, el servicio no está habilitado para iniciar automáticamente

Verificar si un servicio está corriendo sudo service nombreServicio status Iniciar un servicio sudo service nombreServicio start Detener un servicio sudo service nombreServicio stop Habilitar el inicio automático de un servicio sudo update-rc.d nombreServicio defaults Deshabilitar el inicio automático de un servicio sudo update-rc.d nombreServicio remove Listar los servicios corriendo y su puerto sudo ss -tulnp
